# UX検定基礎 学習サイト

`docs/` 配下の教材（試験対策ガイド＋重要語句集）を、ブラウザで「読む・覚える・解く」ことができる学習サイトです。
ビルド不要・依存なしの静的サイトで、`site/index.html` をブラウザで開くだけで動作します。

## 使い方

`site/index.html` をダブルクリックして開くだけです（サーバー不要）。
学習の進捗（読了章・単語の習得状況・問題の成績）は、ブラウザの localStorage に自動保存されます。

### 主な機能

| 機能 | 内容 |
|---|---|
| **ホーム** | 試験までのカウントダウン、学習進捗サマリー、試験概要 |
| **読む** | 本編テキスト全9章。章内目次・読了管理・章末ミニ問題への導線 |
| **単語カード** | 重要語句74語の暗記カード。めくり／「覚えた・要復習」の仕分け／フィルタ／シャッフル（キーボード操作可） |
| **問題演習** | 本番モード模試（100問100分・採点）、即時解説の演習、カテゴリ別／章別、**間違い復習**、カテゴリ別の正答内訳 |
| **検索** | 重要語句・本編・問題を横断検索 |
| **計画・資料** | 試験概要、シラバス網羅表、直前復習メモ、2週間／4週間学習計画 |

ライト／ダークテーマ切替（右上の ◐）、スマホ対応。

## 教材を更新したら

学習コンテンツは `docs/*.md` から自動生成しています。Markdown を編集したら、次を実行してサイトに反映します。

```bash
node site/build.js
```

これで `site/js/data.js`（全コンテンツのデータ）が再生成されます。
ビルド時に「章＝9・各章ミニ問題＝5問・語句＝74・模試＝100問」などの整合性を検証し、欠落があれば警告を出します。

## ファイル構成

```
site/
├── index.html      … サイトの土台（ヘッダ・ナビ・フッタ）
├── css/style.css   … スタイル（ライト/ダーク・レスポンシブ）
├── js/
│   ├── data.js     … docs から自動生成したデータ（直接編集しない）
│   └── app.js      … アプリ本体（ルーター・各画面・クイズエンジン）
└── build.js        … docs/*.md → js/data.js の変換スクリプト
```
